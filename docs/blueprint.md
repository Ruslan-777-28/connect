# **App Name**: ConnectU

## Core Features:

- User Authentication: Enable user registration and login using Firebase Authentication.
- Profile Management: Allow users to create and edit their profiles, including uploading profile pictures to Firebase Storage and saving data to Firestore.
- User Listing: Display a list of all registered users on the 'Home' page, fetching user data from Firestore.
- Profile Viewing: Allow users to view the profiles of other users.
- Firebase Initialization Tool: A tool to generate necessary .env.local configuration settings initialized for Firebase to ensure the application works correctly.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to create a calm and inviting atmosphere.
- Background color: Very light blue (#F0F8FF), a desaturated version of the primary color, for a clean backdrop.
- Accent color: Light purple (#B4A7D6), an analogous color to the primary, for highlights and interactive elements.
- Body and headline font: 'Inter' sans-serif, to ensure clear presentation of profile and user data.
- Use simple, outlined icons for navigation and profile actions.
- Employ a clean and responsive layout, ensuring usability on various devices.
- Incorporate subtle transitions and loading animations to enhance user experience.
